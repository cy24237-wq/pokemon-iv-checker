# Pokemon IV Checker

## 概要
ポケモンSVの個体値を評価するWebアプリです。  
ポケモン名を入力すると、登録されたデータをもとに個体値や種族値を確認でき、育成判断をサポートします。

## 制作背景
ポケモン対戦では、個体値や種族値を確認しながら育成方針を決める必要があります。  
しかし、複数の情報を別々に確認するのは手間がかかるため、必要な情報を1つの画面で確認できるツールを作成しました。

## 使用技術
- HTML
- CSS
- JavaScript
- Node.js
- PokeAPI
- GitHub Pages

## 主な機能
- ポケモン名検索
- 個体値評価
- 種族値データ表示
- JSONデータを用いたポケモン情報管理
- PokeAPIからのデータ取得

## 工夫した点
### 1. APIを活用したデータ取得
PokeAPIを利用し、ポケモンのデータを取得する処理を作成しました。  
手作業でデータを入力するのではなく、JavaScriptで自動取得・JSON化することで、管理しやすい構成にしました。

### 2. ユーザーが使いやすい画面設計
ポケモン名を入力してすぐに評価結果を確認できるようにし、直感的に操作できるUIを意識しました。

### 3. データをJSONで管理
ポケモン情報をJSON形式で管理することで、検索や表示処理を分かりやすく整理しました。
